# おなまえ「あいうえお」(Vue & Firebase)

「おなまえ『あいうえお』」は幼児向けの「ひらがな」学習用アプリです。

![クイズ画面](./quiz-img.png)

表示されるイラストの名前を五十音表から入力する「くいずモード」と、
好きにひらがなを入力して入力された名前のイラストを表示する「じゆうモード」で、
遊びながら、ひらがなを学習できます。

[おなまえ「あいうえお」](https://a-i-u-e-o.web.app/)
